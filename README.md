# ZAP — The Read Before the Action

ZAP is an intent-first football strategy game built around one core idea:

Football is not only about execution. It is about reading the moment before the ball moves.

Instead of directly controlling every pass, shot, or tackle, players face football situations and choose the read they believe will beat the opponent’s read. Pressure, space, support, risk, and timing become the real game.

Every turn in ZAP is a prediction battle:
your read against their read.

The game is designed around short, high-tension football encounters where players must understand the situation, anticipate the opponent’s intent, and commit to a decision before the result resolves through deterministic game logic.

Built on onchain systems, ZAP treats football not as animation-first gameplay, but as a living strategic world driven by state, consequence, and verifiable resolution.

At its core, ZAP is not trying to recreate football control.

It is trying to recreate football thought.
